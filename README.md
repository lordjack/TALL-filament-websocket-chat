## CineBiz Filament Conversations

### Install

```
composer install

```
### Connect Storage link to upload
```
php artisan storage:link
```
**-> Setup DB-Connection in .env file!**

```
php artisan migrate --seed
```

### FE Dependencies

```
npm install
```
```
npm run build
```

### Run App

```
php artisan serve
```

### Run Websocket server

```
php artisan websockets:serve
```
